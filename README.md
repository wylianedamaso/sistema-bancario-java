# Sistema Bancário em Java
## Descrição

Projeto desenvolvido com o objetivo de praticar lógica de programação utilizando Java.  
O sistema funciona no terminal e simula operações bancárias básicas, como consulta de saldo, depósito e saque.

Durante o desenvolvimento foram aplicados conceitos fundamentais da linguagem, incluindo estruturas de repetição, estruturas de decisão e validações para garantir que operações inválidas (como valores negativos ou saldo insuficiente) não sejam realizadas.

## Funcionalidades

- Consultar saldo
- Realizar depósito
- Realizar saque
- Validação de valores inválidos
- Validação de saldo insuficiente
- Menu interativo no terminal

## Conceitos Aplicados

- Variáveis e tipos primitivos (int, double)
- Estrutura de repetição (while)
- Estrutura de decisão (switch)
- Condicionais (if / else)
- Entrada de dados com Scanner

## Tecnologias Utilizadas

- Java
- Intellij IDEA
- Git
- GitHub

## Como Executar

1. Clone este repositório:
   git clone https://github.com/wylianedamaso/sistema-bancario-java.git

2. Compile o arquivo: 
   javac br.com.wyliane.sistemabancario.SistemaBancario.java

3. Execute o programa:
   java br.com.wyliane.sistemabancario.SistemaBancario
